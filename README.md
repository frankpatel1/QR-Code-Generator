# QR Code Generator

## 🔳 Overview

QR Code Generator is a web-based application that enables users to generate QR codes instantly from text, URLs, contact details, or other input data. Built using HTML, CSS, and JavaScript, the project offers a simple and intuitive interface for creating and downloading QR codes directly from the browser.

This project demonstrates practical implementation of JavaScript event handling, API integration, dynamic content rendering, and responsive web design principles.

## ✨ Features

* Generate QR codes instantly
* Support for URLs, text, and custom data
* Real-time QR code preview
* Download generated QR codes
* Responsive and mobile-friendly design
* Simple and intuitive user interface
* Fast client-side processing
* Lightweight and easy to customize

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* QR Code Generation API / Library

## 🚀 Getting Started

### Prerequisites

To run this project, you only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/QR-Code-Generator.git
```

Navigate to the project directory:

```bash
cd QR-Code-Generator
```

Open the application:

```bash
index.html
```

Or simply open `index.html` in your browser.

## 📖 Usage

1. Launch the application.
2. Enter text, a website URL, or other supported data.
3. Click the generate button.
4. View the generated QR code instantly.
5. Download the QR code image for sharing or printing.
